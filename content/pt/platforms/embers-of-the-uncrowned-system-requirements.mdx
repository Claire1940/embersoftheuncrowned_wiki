<Section icon={Shield} title="Requisitos Oficiais de Sistema de Embers of the Uncrowned">

Compreender os **requisitos de sistema de Embers of the Uncrowned** é o primeiro passo para qualquer mercenário que busca retomar a Casa Harborwell. Desenvolvido pela NEXON, este MMORPG de fantasia sombria apresenta ambientes isométricos de alta fidelidade, efeitos de habilidade complexos e chefes de domínio maciços que podem exigir muito de hardware mais antigo. Embora o jogo seja otimizado para processadores multi-core modernos, a transição de um pequeno acampamento para uma cidade vibrante no sistema de assentamento requer VRAM consistente e armazenamento rápido para evitar travamentos durante o streaming de ativos.

**Destaques do Vídeo:**
- Visão geral do mundo isométrico de fantasia sombria e fluxo de combate.
- Demonstração visual das três classes principais: Carrasco, Lâmina Espectral e Tempestário.
- Prévia do sistema de crescimento de assentamento da Casa Harborwell.
- Filmagem de encontros maciços com Chefes de Domínio e mecânicas de raide.

<Section icon={Target} title="Guia de Instalação e Configuração">

Depois de verificar se o seu PC atende aos **requisitos de sistema de Embers of the Uncrowned**, você pode prosseguir com a instalação. A demo lançada em 13 de junho de 2026 é uma entrada separada na loja Steam da página "Em Breve" do jogo completo. Siga estas etapas para garantir uma configuração limpa.

<Steps>
  <Step num="1" title="Localize a Página da Demo">
    Navegue até a loja oficial do Steam e procure especificamente por **Embers of the Uncrowned Demo**. Não apenas adicione o jogo principal à sua lista de desejos; você deve baixar o cliente demo específico para o evento Steam Next Fest.
  </Step>
  <Step num="2" title="Verifique o Espaço em Disco">
    Certifique-se de ter pelo menos 30 GB de espaço livre. Embora o tamanho do download possa ser menor devido à compressão, os arquivos não compactados e futuros patches durante o período da demo exigirão a alocação total.
  </Step>
  <Step num="3" title="Atualize os Drivers Gráficos">
    NVIDIA, AMD e Intel frequentemente lançam drivers "Game Ready" para os principais lançamentos da NEXON. Certifique-se de que seus drivers estejam atualizados para as versões de junho de 2026, no mínimo, para evitar artefatos visuais em ambientes de fantasia sombria.
  </Step>
  <Step num="4" title="Configure o Anti-Cheat NGS">
    No primeiro lançamento, o prompt do Nexon Game Security (NGS) aparecerá. Este é um requisito de nível de kernel para jogar online. Você deve conceder permissões administrativas para que o jogo inicialize seus componentes multiplayer.
  </Step>
</Steps>

| Tipo de Requisito | Detalhe | Ação do Jogador |
| :--- | :--- | :--- |
| **Rede** | Internet Banda Larga | Necessário para todos os modos de jogo |
| **Conta** | Conta Steam | Necessário para demo e jogo completo |
| **Controle** | Suporte para Xbox / DualShock | Opcional, mas recomendado para combate manual |
| **Anti-Cheat** | NGS (nível de kernel) | Instalação obrigatória |

<Callout type="success" title="Pronto para Lançar">
Após seguir estas etapas, você poderá selecionar sua classe — Carrasco, Lâmina Espectral ou Tempestário — e começar sua jornada para retomar o domínio perdido da Casa Harborwell.
</Callout>

<Section icon={Sparkles} title="Informações de Segurança e Anti-Cheat">

Como um MMORPG moderno com elementos de Arena PvP e Raide competitivos, Embers of the Uncrowned emprega medidas rigorosas anti-cheat. O uso do **Nexon Game Security (NGS)** é um padrão para títulos da NEXON, mas vem com implicações específicas de nível de hardware que os jogadores devem estar cientes antes da instalação.

| Recurso | Descrição | Impacto |
| :--- | :--- | :--- |
| **Nível** | Nível de Kernel (Anel 0) | Acesso de alto nível ao sistema |
| **Função** | Varredura de memória / Detecção de macro | Impede vantagens injustas no PvP |
| **Remoção** | Remoção manual necessária | Nem sempre é excluído na desinstalação do Steam |
| **Compatibilidade** | Windows 11 Secure Boot | Pode exigir ajustes no BIOS |

<Callout type="warning" title="Nota Importante sobre a Desinstalação">
Observe que sistemas anti-cheat de nível de kernel como o NGS às vezes permanecem no sistema após uma desinstalação padrão do Steam. Se você encontrar problemas com outros jogos após remover a demo, pode ser necessário usar uma ferramenta de remoção dedicada ou excluir manualmente a pasta de serviço do NGS.
