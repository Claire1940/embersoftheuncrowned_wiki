<Section icon={Swords} title="Основные боевые механики и синергия стихий">

Боевой цикл для **Stormbringer из Embers of the Uncrowned** построен на синергии Ветра и Молнии. В ручной изометрической боевой системе вы не можете просто стоять на месте и колдовать; вы должны использовать механику "Уклонение", которая расходует SP, чтобы избегать смертельных атак боссов, поддерживая свою магическую ротацию.

### Система двойной силы
Stormbringer не просто наносит урон; он контролирует пространство. Навыки Ветра в основном используются для группировки врагов или отталкивания агрессивных юнитов ближнего боя, в то время как навыки Молнии действуют как "добивающий удар", чтобы использовать созданные возможности.

| Тип магии | Основной эффект | Лучше всего использовать для |
| :--- | :--- | :--- |
| **Ветер** | Смещение/Замедление | Контроль толпы, кайтинг и создание дистанции. |
| **Молния** | Высокий взрыв/Шок | Урон боссам, пробитие защиты и критические удары. |

Эслин действует как ваш основной фронтовик. Развивая с ней **Сродство** через эксклюзивные квесты, вы разблокируете навыки компаньона, которые напрямую дополняют ваши билды Stormbringer. Например, Эслин может провоцировать врагов, удерживая их в вашем **Статическом поле** для максимального урона.

### Борьба с боссами Домена
Боссы Домена требуют от вас возвращения земель для Дома Харборвелл. В роли Stormbringer ваша задача — оставаться на максимальной дистанции и использовать навыки Ветра, чтобы отталкивать "аддов" (более мелких врагов), сосредоточив Молнию на боссе.

| Тип босса | Стратегия Stormbringer | Сложность |
| :--- | :--- | :--- |
| **Полевой босс** | Кайтить по кругу; использовать Ходьбу по ветру, чтобы избегать АОЕ-ударов. | Легко |
| **Босс Домена** | Сосредоточиться на прерывании кастующихся навыков с помощью Шока Молнии. | Средне |
| **Рейдовый босс** | Координировать действия с группой; сохранять взрывной урон для окон Оглушения. | Сложно |
