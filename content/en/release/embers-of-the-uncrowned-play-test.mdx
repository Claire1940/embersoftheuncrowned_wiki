The combat in Embers of the Uncrowned emphasizes manual control, precise evasion, and tactical skill usage. Understanding the core combat loop is essential for survival and success against the various enemies and bosses.

| Combat Element      | Description                                                                 | Player Action                                             |
| :------------------ | :-------------------------------------------------------------------------- | :-------------------------------------------------------- |
