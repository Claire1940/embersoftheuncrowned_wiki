<Section icon={Gem} title="Progression and Skill Gems">
Progression in Embers of the Uncrowned extends beyond basic leveling, integrating active and passive skills, Skill Gems, gear enhancement, and bloodline awakening. Each of the **Embers of the Uncrowned classes** has 28-29 unique skills, allowing for deep customization and varied combat approaches.

| Progression Layer   | Description                                            | Impact on Class Build                                   |
| :------------------ | :----------------------------------------------------- | :------------------------------------------------------ |
| **Active Skills**   | Moment-to-moment combat choices, direct abilities      | Defines immediate combat rhythm and damage output       |
| **Passive Skills**  | Supports class identity, survivability, damage style   | Enhances core strengths, provides utility and defenses   |
| **Skill Gems**      | Long-term character growth layer, modify abilities     | Adds unique effects, scales late-game power, opens new combos |
| **Gear Enhancement**| Improves equipment stats and bonuses                   | Direct power increase, essential for scaling damage/defense |
| **Bloodline Power** | Inherited power, connects to late-game progression     | Unlocks powerful abilities and passive bonuses, ties to lore |
